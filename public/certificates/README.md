# Certificates Folder

Place your certificate files here. Supported formats: PDF, PNG, JPG, WebP.

## Expected filenames:

| Certificate                    | Filename                  |
|-------------------------------|---------------------------|
| MERN Stack Development         | mern-stack.pdf            |
| C/C++ Programming              | cpp-programming.pdf       |
| Java Programming               | java-programming.pdf      |
| Data Structures & Algorithms   | dsa.pdf                   |
| Blockchain Builder Certificate | blockchain.pdf            |
| Internship Completion          | internship.pdf            |

> You can also use image files (.png, .jpg). Just update `certificateUrl` in `src/utils/data.js` to match the filename.
